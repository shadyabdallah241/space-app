# Space Explorer 🚀

Space Explorer is a Flutter application designed to provide an interactive and educational experience exploring our solar system. Users can learn about different planets, their characteristics, and fascinating facts.

## ✨ Features

*   **Secure Login:** Start your journey with a dedicated login screen.
*   **Planet Catalog:** Browse through all the planets in our solar system, including the Sun.
*   **Detailed Insights:** Each planet has a dedicated details page featuring:
    *   Informative descriptions.
    *   Key statistics: Distance from Sun, Orbital Period, Radius, Mass, Gravity, and Surface Area.
*   **Immersive UI:** A visually appealing interface with high-quality planet imagery and custom widgets.

## 🛠️ Project Structure

The project follows a clean and modular structure:

*   `lib/models/`: Contains the data models (e.g., `Planet`).
*   `lib/data/`: Static data for the planets.
*   `lib/screens/`: UI screens for Login, Home, and Details.
*   `lib/widgets/`: Reusable custom UI components like `RedButton` and `ArrowButton`.
*   `lib/components/`: Specific components like the custom `AppBar`.

## 🚀 Getting Started

### Prerequisites

*   Flutter SDK installed on your machine.
*   An IDE (Android Studio, VS Code, etc.).
*   An emulator or physical device for testing.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/SpaceTask.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd SpaceTask
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the application:**
    ```bash
    flutter run
    ```

## 📦 Dependencies

*   `cupertino_icons`: For iOS-style icons.
